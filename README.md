# Harshith R â€“ Portfolio (React + Tailwind + Framer Motion)

A modern, fully responsive portfolio for an AI Engineer with a futuristic dark theme, smooth animations, and clean structure. Built with React (Vite), Tailwind CSS, and Framer Motion.

## Tech Stack
- React (Vite)
- Tailwind CSS (dark mode via `class`)
- Framer Motion
- react-icons, react-scroll
- react-simple-typewriter
- react-tsparticles (particles background)

## Features
- Sticky navbar with active section highlight and smooth scroll
- Hero with typewriter roles, CTA buttons, and particles background
- About, Skills (progress bars), Projects (animated cards), Education (timeline)
- Contact form with validation and mock success animation + social icons
- Footer with quick links and year
- Dark/Light mode toggle (persisted)
- Scroll-to-top button
- SEO-ready (meta tags, title, favicon)

## Getting Started

```bash
# Install dependencies
npm install

# Run in dev mode
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Customize Content
Edit files in `src/components/*` and section data arrays inside those components (e.g., skills, projects, timeline). Replace placeholder links (resume, social, demos) and the email in `Contact.jsx`.

## Notes
- Fonts: Poppins/Inter for headings, Open Sans for body (loaded in `index.html`).
- Tailwind config: `tailwind.config.cjs` with extended colors and shadows.
- Particles use `react-tsparticles`; you can swap to `@tsparticles/react` later if desired.
